# Fetch API – Vanilla JavaScript

This repository contains a simple practice project built while learning the **Fetch API** in JavaScript. It demonstrates how to make GET and POST requests, handle JSON data, and update the DOM dynamically without reloading the page.

## 🚀 Features
- Fetching JSON data from an API
- Displaying fetched data in the browser
- Submitting form data using a POST request
- Handling responses and converting them to JSON
- Clean, beginner‑friendly JavaScript examples

## 📂 Project Files
- `index.html` — Main page with form and output area  
- Any supporting files used for testing (e.g., JSONPlaceholder)
